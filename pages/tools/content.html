Tools: Youtube-dl

  1. Download youtube-dl.exe
  2. Install Visual C++ 2010 x86
  3. Download FFmpeg release essentials
  4. Create Youtube-dl directory in C
  5. Copy inside Youtube-dl directory:
    • youtube-dl.exe
    • ffmpeg/bin content
    • youtube-dl.bat
    • icon.ico
    • Youtube-dl.lnk
  6. Put C:\Youtube-dl in PATH
  7. Put youtube-dl.conf in C:\Users\James
  8. Assign icon to Youtube-dl.lnk
  9. Put Youtube-dl.lnk in Start menu
